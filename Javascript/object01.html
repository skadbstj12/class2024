01. at()

지정한숫자(index)를 받아, 문자열에서 해당되는 인덱스의 요소를 반환합니다. 대소문자를 구별합니다. 문자열의 길이보다 큰 인덱스를 지정하면 undefind를 반환합니다. 음수인 경우 문자열의 오른쪽에서부터 인덱스를 셉니다.

{
"javascript reference".at()       //j
"javascript reference".at(0)      //j
"javascript reference".at(1)      //a
"javascript reference".at(2)      //v
"javascript reference".at(3)      //a
"javascript reference".at(4)      //s
"javascript reference".at(5)      //c
"javascript reference".at(-1)     //e
"javascript reference".at(-2)     //c
"javascript reference".at(-3)     //n
"javascript reference".at(-4)     //e
"javascript reference".at(-5)     //r
"javascript reference".at(100)    // undefined
}
결과 확인하기
false

02. includes()

includes() 메소드는 문자열에 특정 문자열 또는 문자가 포함되어 있는지를 확인하는 JavaScript의 메소드입니다. 부분 문자열이 포함되어 있는 경우 true를 반환 그렇지 않은 경우 false를 반환합니다.

{
const str = 'Hello, world!';

console.log(str.includes('world'));
console.log(str.includes('foo'));
}
결과 확인하기
true
false

03. indexOf()

indexOf() 메소드는 문자열에서 지정된 문자열 또는 문자의 첫 번째 발생 위치를 반환합니다. 문자열에 지정된 문자열 또는 문자가 포함되어 있지 않으면 -1을 반환합니다.

{
const str = 'Hello, world!';

console.log(str.indexOf('world'));
console.log(str.indexOf('foo'));
}
결과 확인하기
7
-1

04. match()

정규 표현식과 문자열을 비교하여 일치하는 부분을 찾아내는 JavaScript의 문자열 메서드입니다. 문자열에서 첫 번째로 일치하는 부분을 찾아내며, 일치하는 부분이 없으면 null을 반환합니다. (\w+)는 알파벳 문자나 숫자로 이루어진 하나 이상의 문자열을 의미합니다.

{
const str = "Hello, my name is John Doe.";
const pattern = /name is (\w+)/;

const result = str.match(pattern);

console.log(result);
}
결과 확인하기
["name is John", "John"]

05. replace()

문자열 내에서 특정 패턴을 찾아 지정된 다른 문자열로 대체하는 JavaScript의 문자열 메서드입니다. 문자열에서 일치하는 모든 패턴을 변경할 수 있습니다. 첫 번째로 일치하는 패턴을 대체합니다.

{
const str = "I like bananas. Do you like bananas?";
const replacedStr = str.replace("bananas", "apples");

console.log(replacedStr);
}
결과 확인하기
"I like apples. Do you like bananas?"

06. search()

문자열 내에서 특정 패턴이나 문자열을 찾아 해당 위치(인덱스)를 반환하는 JavaScript의 문자열 메서드입니다. 첫 번째로 발견된 패턴의 인덱스만을 반환하며, 모든 발견된 패턴의 인덱스를 반환하지는 않습니다. 찾고자 하는 패턴이 없으면 -1을 반환합니다.

{
const str = "Hello, world!";
const position = str.search("world");

console.log(position);
}
결과 확인하기
7

07. slice()

문자열이나 배열에서 지정한 위치의 부분을 추출하여 새로운 문자열이나 배열을 반환하는 JavaScript의 내장 함수입니다. 메서드는 원본 문자열이나 배열을 변경하지 않고, 추출된 부분을 포함하는 새로운 문자열이나 배열을 반환합니다. 

{
const str = "Hello, world!";
const slicedStr = str.slice(7);

console.log(slicedStr);
}
결과 확인하기
"world!"

08. substring()

문자열에서 지정한 시작 위치부터 끝 위치까지의 부분 문자열을 추출하여 반환하는 JavaScript의 내장 함수입니다. 시작 위치와 끝 위치를 지정하여 부분 문자열을 추출합니다. 끝 위치를 지정하지 않으면 문자열의 끝까지를 추출합니다. 또한, 두 번째 매개변수로 지정한 끝 위치는 포함되지 않습니다. 

{
const str = "Hello, world!";
const subStr = str.substring(7, 12);

console.log(subStr);
}
결과 확인하기
"world"

09. split()

.split()

문자열을 지정된 구분자를 기준으로 나누어 배열로 반환하는 JavaScript의 내장 함수입니다. 원본 문자열을 변경하지 않고, 구분자를 기준으로 문자열을 나눈 후 각 부분을 배열의 요소로 저장하여 반환합니다. 
{
const str = "apple, banana, orange";
const fruits = str.split(", ");

console.log(fruits);
}
결과 확인하기
["apple", "banana", "orange"]

10. trim()

문자열의 양 끝에서 공백을 제거한 새로운 문자열을 반환하는 JavaScript의 내장 함수입니다. 공백은 여러 가지 유형의 공백 문자(스페이스, 탭, 개행 등)을 포함합니다. 중간에 있는 공백은 제거하지 않습니다. 만약 중간에 있는 공백도 제거하고 싶다면 다른 방법을 사용해야 합니다. 원본 문자열을 변경하지 않고 새로운 문자열을 반환합니다. 따라서 trim() 메서드를 호출하더라도 원본 문자열은 변경되지 않습니다.

{
const str = "   Hello, world!   ";
const trimmedStr = str.trim();

console.log(trimmedStr);
}
결과 확인하기
"Hello, world!"
