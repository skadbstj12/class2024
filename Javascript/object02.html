01. find()

주어진 판별 함수를 만족하는 배열의 첫 번째 요소를 반환하는 JavaScript의 배열 메서드입니다. 요소를 찾지 못하면 undefined를 반환합니다. 조건을 만족하는 첫 번째 요소만을 반환하며, 그 이후의 요소는 고려하지 않습니다. 조건을 만족하는 모든 요소를 찾고 싶다면 filter() 메서드를 사용해야 합니다.

{
const numbers = [10, 20, 30, 40, 50];

const found = numbers.find(function(element) {
    return element > 25;
});

console.log(found);
}
결과 확인하기
30

02. forEach()

배열의 각 요소에 대해 주어진 함수를 실행하는 JavaScript의 배열 메서드입니다. 배열의 각 요소에 대해 반복하면서 지정된 함수를 호출합니다.

{
const numbers = [1, 2, 3, 4, 5];

numbers.forEach(function(element) {
    console.log(element);
});
}
결과 확인하기
1
2
3
4
5

03. map()

배열 내의 모든 요소에 대해 주어진 함수를 호출한 결과를 모아서 새로운 배열을 반환하는 JavaScript의 배열 메서드입니다. 원본 배열을 변경하지 않으며, 변환된 배열을 반환합니다.

{
const numbers = [1, 2, 3, 4, 5];

const squaredNumbers = numbers.map(function(element) {
    return element * element;
});

console.log(squaredNumbers);
}
결과 확인하기
[1, 4, 9, 16, 25]

04. includes()

includes() 메소드는 문자열에 특정 문자열 또는 문자가 포함되어 있는지를 확인하는 JavaScript의 메소드입니다. 부분 문자열이 포함되어 있는 경우 true를 반환 그렇지 않은 경우 false를 반환합니다.

{
const str = 'Hello, world!';

console.log(str.includes('world'));
console.log(str.includes('foo'));
}
결과 확인하기
true
false

05. join()

배열의 모든 요소를 하나의 문자열로 결합하여 반환하는 JavaScript의 배열 메서드입니다. 각 요소 사이에는 지정한 구분자를 넣어줄 수 있습니다. 만약 구분자를 지정하지 않으면 기본적으로 쉼표가 사용됩니다. 만약 배열의 요소가 객체인 경우, 각 객체는 자동으로 toString() 메서드를 통해 문자열로 변환됩니다.

{
const fruits = ["apple", "banana", "orange"];
const result = fruits.join(", ");

console.log(result);
}
결과 확인하기
"apple, banana, orange"

06. indexOf()

indexOf() 메소드는 문자열에서 지정된 문자열 또는 문자의 첫 번째 발생 위치를 반환합니다. 문자열에 지정된 문자열 또는 문자가 포함되어 있지 않으면 -1을 반환합니다.

{
const str = 'Hello, world!';

console.log(str.indexOf('world'));
console.log(str.indexOf('foo'));
}
결과 확인하기
7
-1

07. push()

배열의 끝에 하나 이상의 요소를 추가하는 JavaScript의 배열 메서드입니다. 배열의 길이를 변경하고, 새로운 요소를 배열의 끝에 추가합니다.

{
const fruits = ["apple", "banana"];
fruits.push("orange");

console.log(fruits);
}
결과 확인하기
["apple", "banana", "orange"]

08. slice()

문자열이나 배열에서 지정한 위치의 부분을 추출하여 새로운 문자열이나 배열을 반환하는 JavaScript의 내장 함수입니다. 메서드는 원본 문자열이나 배열을 변경하지 않고, 추출된 부분을 포함하는 새로운 문자열이나 배열을 반환합니다. 

{
const str = "Hello, world!";
const slicedStr = str.slice(7);

console.log(slicedStr);
}
결과 확인하기
"world!"

09. splice()

배열의 내용을 변경하고, 삭제된 요소를 포함하는 새로운 배열을 반환하는 JavaScript의 배열 메서드입니다. 배열의 특정 위치에서 요소를 추가, 삭제, 또는 교체할 수 있습니다. 세 가지 매개변수 시작 인덱스: 요소를 추가, 삭제, 또는 교체할 시작 위치를 지정합니다. 삭제할 요소의 개수: 시작 위치부터 삭제할 요소의 개수를 지정합니다. 추가할 요소: 선택적으로 추가할 요소를 지정합니다. 이러한 요소는 시작 위치부터 추가됩니다.

{
                    const fruits = ["apple", "banana", "orange", "grape"];

                    const deletedItems = fruits.splice(2, 1);
                    
                    console.log(fruits);
                    console.log(deletedItems);
}
결과 확인하기
["apple", "banana", "grape"]
["orange"]

10. sort()

배열의 요소를 정렬하여 해당 배열을 변경하는 JavaScript의 배열 메서드입니다. 본적으로 배열은 Unicode 코드 포인트에 따라 문자열로 변환되어 정렬됩니다. 정렬은 기본적으로 오름차순(ascending)으로 이루어집니다.

{
const fruits = ["banana", "apple", "orange"];

fruits.sort();

console.log(fruits);
}
결과 확인하기
["apple", "banana", "orange"]
