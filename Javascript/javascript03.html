01. 함수 : 선언적 함수

선언적 함수(Declarative Function)는 JavaScript에서 함수를 선언하는 기본적인 방법 중 하나입니다. 이 방법은 가장 간단하고 직관적인 방식으로 함수를 정의할 수 있도록 해줍니다. 

{
function func(){
console.log("01. 함수가 실행되었습니다.");
} func();
}

02. 함수 : 익명 함수

익명 함수(Anonymous Function)는 이름이 없는 함수를 가리킵니다. 즉, 함수를 정의할 때 함수 이름을 붙이지 않고 만드는 것을 말합니다. 익명 함수는 주로 다른 함수의 매개변수로 전달되거나, 변수에 할당되는 등의 상황에서 사용됩니다. 

{
const func = function () {
console.log("02. 함수가 실행되었습니다.");
}
func();
}

03. 함수 : 매개변수 함수

매개변수 함수(Parameter Function)는 다른 함수의 매개변수로서 전달되는 함수를 가리킵니다. 

{
// 선언적 함수
function func(str) {
console.log(str);
}func("03. 함수가 실행되었습니다.");

// 익명 함수
const func1 = function (str) {
console.log(str);
}func1("03. 함수가 실행되었습니다.")
};

04. 함수 : 리턴값 함수

리턴값 함수(Return Value Function)는 함수가 호출될 때 값을 반환하는 함수를 가리킵니다. JavaScript에서 함수는 return 키워드를 사용하여 값을 반환할 수 있습니다. 

// 선언적 함수
function func() {
return "04. 함수가 실행되었습니다."
}console.log(func());

// 익명 함수
const func1 = function () {
return "04. 함수가 실행되었습니다."
}console.log(func1());

매개변수와 리턴값을 가지는 함수는 입력값을 받아들이고 처리한 후에 결과를 반환하는 함수를 가리킵니다. 이러한 함수는 입력값을 받아들여 처리하고, 처리한 결과를 다시 호출한 곳으로 반환합니다.

08. 화살표 함수 : 매개변수 함수

화살표 함수(arrow function)는 JavaScript에서 함수를 선언하는 새로운 문법적인 형태입니다. 이 방법은 함수를 더 간결하게 작성할 수 있도록 도와주는데, 주로 함수 표현식의 형태로 사용됩니다. 기존의 함수 선언 방식에 비해 간단하고 명확하게 표현할 수 있으며, 특히 단일 표현식을 반환하는 간단한 함수를 정의할 때 유용합니다. 

// 선언적 함수
func = (str) => {
console.log(str);

// 익명 함수

09. 화살표 함수 : 리턴값 함수

화살표 함수(arrow function)는 JavaScript에서 함수를 정의하는 새로운 방식 중 하나로, 함수를 간결하게 정의할 수 있도록 도와줍니다. 이는 주로 함수 표현식의 형태로 사용되며, 함수를 더 읽기 쉽고 이해하기 쉽게 만들어줍니다. 화살표 함수는 일반 함수와는 다르게 this 컨텍스트가 상위 스코프를 유지하고, 함수 본문이 한 줄로 간단한 경우에는 중괄호와 return 키워드를 생략하여 간결하게 표현할 수 있습니다. 이러한 특성들로 인해 코드를 더 간결하고 가독성 있게 작성할 수 있습니다. 

// 선언적 함수
func = () => {
return "09. 함수가 실행되었습니다."
}console.log(func());

// 익명 함수
const func1 = () => {
return "09. 함수가 실행되었습니다."
}console.log(func1());

10. 화살표 함수 : 매개변수 + 리턴값 함수

화살표 함수(arrow function)는 JavaScript에서 함수를 정의하는 새로운 형식 중 하나입니다. 이는 기존의 함수 정의 방식인 function 키워드를 대체하면서, 코드를 간결하게 작성할 수 있도록 도와줍니다. 이를 통해 함수를 간결하게 정의할 수 있으며, 자바스크립트 코드를 보다 간단하고 가독성 있게 만들어줍니다. 함수의 매개변수와 반환값을 정의하는 방식은 일반적인 함수와 동일하지만, 문법적으로 더 간결하게 표현됩니다. 이러한 특성으로 인해 화살표 함수는 모던 자바스크립트 코드에서 널리 사용되며, 더 나은 코드 작성 경험을 제공합니다. 

// 익명 함수(화살표 함수) + 매개변수 + 리턴값
const func = (str) => {
return str;
}console.log(func("10. 함수가 실행되었습니다."));

// 매개변수가 하나이면 괄호 생략 가능
const func2 = str => {
return str;
}console.log(func2("10. 함수가 실행되었습니다."));

// 리턴 생략
const func3 = str => str;
console.log(func3("10.함수가 실행되었습니다."));

// 선언적 함수(가독성X)
func4 = str => str;
console.log(func4("10.함수가 실행되었습니다."));
결과 확인하기
10. 함수가 실행되었습니다.

19. 함수 유형 : 객체 리터럴 함수

이 코드는 객체 리터럴 함수를 사용하여 메서드를 추가한 후, 두 개의 객체를 생성하여 해당 메서드를 호출하여 결과를 출력합니다. 객체 리터럴을 통해 Func.prototype에 result1과 result2 메서드를 추가하여 각각의 객체에서 호출할 수 있게 합니다. 

{
// 함수 정의
function Func(num, str) {
this.num = num;
this.str = str;
}

//메서드 추가
Func.prototype = {
result1: function () {
console.log(`${this.num}. ${this.str}`);
},
result2: function () {
console.log(`${this.num}. ${this.str}`);
}
}

//인스턴스 생성
const info1 = new Func(23, "함수가 실행되었습니다");
const info2 = new Func(24, "함수가 실행되었습니다");

// 호출
info1.result1();
info2.result2();
}
결과 확인하기
23. 함수가 실행되었습니다. 24. 함수가 실행되었습니다.
