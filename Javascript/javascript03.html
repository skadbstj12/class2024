01. 함수 : 선언적 함수

이 코드는 JavaScript로 작성된 함수를 정의하고 호출하는 단순한 프로그램입니다. 함수가 실행되면 콘솔에 "01. 함수가 실행되었습니다."라는 메시지가 출력됩니다. 

{
let x = 100;
let y = 200;
let z = "javascript";

console.log(x);
console.log(y);
console.log(z);
}
결과 확인하기
100
200
javascript
