05. 배열: 데이터 불러오기 : for()문

이 코드는 JavaScript로 작성되었으며, 배열을 반복하면서 각 요소를 출력하는 역할을 합니다. 

{
const arr = [100, 200, 300, 400, 500, 600, 700, 800, 900];

console.log("05. 배열: 데이터 불러오기 : for()문");

06. 배열: 데이터 불러오기 : forEach()

이 코드는 JavaScript로 작성된 배열의 각 요소를 반복하면서 출력합니다. 배열에 저장된 각 값은 forEach() 메서드를 사용하여 요소를 콘솔에 출력합니다. 

{
const arr = [100, 200, 300, 400, 500, 600, 700, 800, 900];

console.log("06. 배열: 데이터 불러오기 : forEach()");
arr.forEach(function (element) {
console.log(element);
});
}

07. 배열: 데이터 불러오기 : for on

for of 반복문은 배열을 순회하면서 요소를 가져오는 작업에서 간결하고 직관적인 문법을 제공하므로 코드를 더 쉽게 이해하고 유지보수할 수 있습니다. 이는 코드의 가독성을 향상시키고 실수를 줄일 수 있습니다. 

{
const arr = [100, 200, 300, 400, 500, 600, 700, 800, 900];

console.log("07. 배열: 데이터 불러오기 : for on");
for (let element of arr) {
console.log(element);
}
}

08. 배열: 데이터 불러오기 : for in

이 코드는 JavaScript로 작성된 배열을 반복하면서 각 요소를 출력합니다. 코드는 for in 반복문을 사용하여 배열의 인덱스를 순회하고, 각 인덱스를 이용하여 배열의 요소를 가져와서 콘솔에 출력합니다. 

{
const arr = [100, 200, 300, 400, 500, 600, 700, 800, 900];

console.log("08. 배열: 데이터 불러오기 : for in");

}
